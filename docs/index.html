javascript プログラミング講座

概要

高校生向けにプログラミングの練習ができるページを作りました。

練習問題があり、実際にコードを書いて確認できます。

分かりやすさを優先し、言語仕様の詳細には立ち入りません。

注意事項

ipadを使用している場合：設定->一般->キーボードのスマート句読点をoffにしてください。onのままだと"が自動的に全角になりsyntax errorが発生します。

google chromeを使用している場合：練習問題のテキストボックスで赤い波線が邪魔なときは、ウインドウの右上…->設定->言語->スペルチェックをoffしてください。

第3回 練習問題（初級）
